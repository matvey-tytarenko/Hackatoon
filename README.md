# Installation & Usage

## Backend

```
cd backend
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
```
Run with ``python3 server.py`` in **backend** folder

## Frontend

```
cd frontend
npm install
```

Run with ``npm run dev`` in **frontend** folder
